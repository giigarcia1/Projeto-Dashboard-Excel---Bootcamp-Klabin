# 📊 Projeto Dashboard Excel - Bootcamp Klabin
📌 Este projeto foi desenvolvido como parte de um desafio do bootcamp, com o objetivo de implementar um **dashboard de vendas** para análise comercial, utilizando o Excel.

A proposta foi transformar dados brutos em informações visuais claras e úteis.

### 🎯 Objetivo
Criar um painel que permita:
- Análise eficaz do desempenho de vendas
- Tomada de decisões baseada em dados
- Acompanhar o faturamento total

### 📈 Análises Apresentadas
- Total de assinaturas do EA Play
- Total de assinaturas do Minecraft Season Pass
- Faturamento total de todas as assinaturas agregadas
- Faturamento total de renovações e não renovações

### 📂 Estrutura do Repositório
- Base de dados: base.xlsx
- Dashboard finalizado: dashboard_xbox_finalizado.xlsx

### ▶️ Como Visualizar?
1. Faça o download do arquivo;
2. Abra na ferramenta correspondente (Excel).
